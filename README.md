# Nomster - A Yelp Clone <br />
https://nomster-karly.herokuapp.com/<br />
This application integrates the Google Map API and includes features like user comments, star ratings, image uploading and user authentication. HTML, CSS, RUBY, RAILS, ALGORITHMS, AWS. <br />
